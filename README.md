# 简易绘图板（MFC + GDI）

## 一、项目简介

 **项目二：简易绘图板（GDI + 撤销 / 重做）**。
基于 **MFC Doc/View 架构** 实现一个基础的 Windows 桌面绘图程序，支持常见图形绘制及操作历史管理。

---

## 二、功能说明

*  基本绘图功能

  * 直线、矩形、椭圆、自由曲线
  * 鼠标左键按下开始绘制，移动预览，松开完成

*  绘图属性

  * 画笔颜色选择
  * 画笔粗细设置（对话框）

*  撤销 / 重做

  * 基于 Command 思想
  * 支持多步 Undo / Redo

*  文件管理

  * 项目内绘图数据保存与加载（自定义格式）

---

## 三、开发环境

* 操作系统：Windows 10 / 11 x64
* 开发工具：Visual Studio 2022
* 框架：MFC（Doc/View）
* 图形库：GDI / GDI+

---

## 四、项目结构说明

```
MFC_draw/
├── MFC_drawApp        // 应用入口
├── MainFrm            // 主框架窗口
├── MFC_drawDoc        // 文档类，管理绘图数据
├── MFC_drawView       // 视图类，处理绘制与鼠标事件
├── CSetPenSizeDialog  // 画笔粗细设置对话框
├── AICommonLog        // AI 使用日志辅助类
├── ai_logs           // AI 使用日志（Prompt / Response / Diff）
└── README.md
```

---

## 五、设计说明（简要）

* 使用 **MFC Doc/View 架构** 解耦数据与显示
* 绘图操作封装为独立对象，便于扩展与维护
* GDI 资源统一管理，避免资源泄漏
* 撤销/重做采用命令式管理思路
